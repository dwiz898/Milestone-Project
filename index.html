My Portfolio

Jubilee Austen Project
HTML 5 & CSS Website

My first large project I completed combining my skills in HTML & CSS. This project demonstrates the basic knowledge I acquired using a design comp to create the layout of a site. HTML & CSS best practices are followed, in addition to incorporating many common elements such as images, navigation and social links and utilizing the box model.

Rogue Pickings Website
HTML & CSS from Photoshop Design Comp

The capstone project for my HTML & CSS learning. This project allowed me to demonstrate my ability to code a website using only a CSS boilerplate and design comp without annotations. I was able to reinforce and expand on the skills and techniques I had learned during the Jubilee project to create a site with a different layout.

Unplugged Retreat Website
HTML 5 & CSS Website

My current ‘magnum opus’ that is the accumulation of my ongoing knowledge in HTML & CSS, as well as my first entirely responsive website. In addition to being responsive across devices, this project also incorporates entirely separate ‘About’ and ‘FAQ’ pages with their own unique content. Elements such as icons, web fonts, input forms, iframes and links are fully developed and realized here.

About Me

I am passionate about the web.

Hello, my name is Dan! Like many others who go into web development, it was not what I originally studied in university. Although I still ended up working in an IT-related field, it was not something I was truly interested in or passionate about. My career change decision came after much introspection about what would make me happy as well as keep me challenged.

I'm always growing!

Outside of coding, my passions include hiking (beautiful 14ers when I can), karate, skiing, video games, sci-fi, card collecting and just about anything that piques my interest (which is a lot). I love to explore new hobbies and learn new things constantly! I can't wait to be a creative partner with you!
